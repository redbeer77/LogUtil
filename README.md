# LogUtil
Log Example in C#

Can configure route and name of log in app.config
```
  <Rutes>
    <add key="Log" value="D:\\" />
  </Rutes>
  <Names>
    <add key="Log" value="Log.log" />
  </Names>
```
Result Example:
```
Init 19/10/2018 18:24:39
	Example Text
	Total: 00:00:00.0679608
End 19/10/2018 18:24:39
```
